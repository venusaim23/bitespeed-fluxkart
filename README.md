# bitespeed-fluxkart
Identify customers across multiple purchases with common mobile and email

## Node Versions Used
- Node - 20.15.0
- NPM - 10.7.0
- NVM - 0.39.7

## Minimum requirements
- Node - >=18

## Run the server locally:

- Install dependencies: `npm install`
- Run the server: `npm run start`
